Contribute
==========

RADIS is an open-source project, and therefore anyone can contribute. 
See the `list of contributors <https://github.com/radis/radis/graphs/contributors>`__ 
on GitHub. 

A good way to contribute is to start from the  
`GitHub opened issues <https://github.com/radis/radis/issues>`__.
For a First Time contribution, you can tackle one of the Issues displayed as
`Good First Issue <https://github.com/radis/radis/contribute>`__.  

You can also see the `Q&A forum <https://groups.google.com/forum/#!forum/radis-radiation>`__ 
or join the community chat:

    :target: https://gitter.im/radis-radiation/community
    :alt: Gitter
